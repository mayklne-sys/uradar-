AudioRadarFFT

Чирп (18→16 кГц) → микрофон → полосовой фильтр → FFT‑корреляция → SNR → дистанция. CameraX как видоискатель, вибро при уверенном пике.

Скачать проект (ZIP) Инструкция по сборке
Стабильность
BPF x2 (16–18 кГц) + minLag≈3 мс отсекают прямую наводку и шум.
Скорость
FFT‑корреляция O(N log N), переиспользование буферов снижает GC.
Фидбек
SNR по коррелограмме, вибро >6 дБ, CameraX‑превью.

Сборка

Android Studio

  1. Распакуйте AudioRadarFFT_Complete_Project.zip.
  2. File → Open… → выберите папку проекта.
  3. Дождитесь синхронизации Gradle.
  4. Build → Build APK(s) → app/build/outputs/apk/debug/app-debug.apk

CLI

./gradlew assembleDebug
./gradlew assembleRelease   # релиз, требует подписания

Требования: Android SDK 34, JDK 17, Kotlin 1.9, AGP 8.5.
